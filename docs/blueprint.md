# **App Name**: Sales Progress Tracker

## Core Features:

- Sales Data Input: Allow users to input sales figures for tracking.
- Progress Calculation: Calculate the sales progress based on target values.
- Horizontal Bar Visualization: Visualize the sales progress using a horizontal bar with stage indicators.
- Stage Indication: Indicate stages on the horizontal bar using color codes: Red (0-1000), Blue (1001-1500), Yellow (1501 -2500) and Green above 2501 to indicate higher stages. AI tool that learns acceptable ranges for stages to incorporate them into the visualization automatically. It incorporates relevant data from market benchmarks and suggests revised sales targets, if necessary, for realistic progress tracking.
- Webpages Generation: Automatically generate a webpage to present these progress. Webpage all the Red, Blue, Yellow, Green and Sales numbers are sent a query parameters. Based on the state of sales, we would show the incentive multiplier for Red 0.75 %, Blue 0.90%, Yellow 1.20% and Green 1.40%

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and stability, complementing the business-oriented nature of the application.
- Background color: Very light gray (#F0F0F3), providing a neutral backdrop that keeps the focus on the sales data.
- Accent color: Teal (#009688) to highlight key performance indicators (KPIs) and actionable items.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text, providing a clean, readable interface.
- Use clear, business-like icons to represent different sales metrics and actions.
- Layout: Prioritize clarity with a clean, structured design that makes data easy to find and interpret at a glance.
- Subtle transitions to highlight changes in sales data, and keep it fresh and modern
